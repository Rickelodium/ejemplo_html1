HTML

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el código estándar que se utiliza para estructurar y desplegar una página web y sus contenidos.

Qué es HTML

Más que un lenguaje de programación, es un lenguaje de marcado. Esto significa que utiliza un sistema de etiquetas (como bloques de construcción) para decirle al navegador web cómo debe mostrar el texto, las imágenes y otros elementos multimedia.

Historia de HTML

Fue inventado por Tim Berners-Lee en 1991. En sus inicios, era muy simple y solo permitía compartir documentos de texto estáticos entre científicos. Con el paso de los años, fue evolucionando e incorporando nuevas etiquetas para hacer la web más visual e interactiva.

HTML5

HTML5 es la versión más reciente y moderna de este lenguaje. Introdujo nuevas etiquetas semánticas (que dan más significado a la estructura), soporte nativo para reproducir audio y video sin necesidad de plugins, y mejoras para hacer que las páginas funcionen mejor en dispositivos móviles.
